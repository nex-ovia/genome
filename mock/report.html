nexovia / project genome

What it is

A single source of truth for the whole project

Not a throwaway diagram — one editable, version-controlled file that humans and coding agents both read and write, kept live on every commit and PR. Three things make it trustworthy:

🧬

One genome, three sources

The same nexovia.toml is generated from existing code, a document, or a plain prompt — and rendered to this HTML for humans.

🔍

Agent-written, human-validated

The scanner and AI agents keep it current on every commit — each field records who wrote it (scan, agent, or human) and how sure. A human edit is a sticky guardrail agents can never overwrite.

♻️

Live, not a snapshot

On each commit/PR it scans only what changed, updates the genome incrementally, and regenerates this page so understanding never drifts.
